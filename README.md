## 🚀️ Executando os projetos

1. Clone o repositório: 
```bash
git clone https://github.com/FabioFiorita/C214-Lab.git && cd C214-Lab
```
2. Entre na aula escolhida

3. Instale todas as depencências
```bash
npm install
```

4. Rode o projeto
```bash
npm start
```
